add32

Add two 32-bit numbers

Input:
(hl), (de) - 32-bit numbers
Output:
(hl) = (hl) + (de)
Destroyed:
none

adc32

Add two 32-bit numbers and the carry bit

Input:
(hl), (de) - 32-bit numbers
Output:
(hl) = (hl) + (de) + cf
Destroyed:
none

sub32

Subtract two 32-bit numbers

Input:
(hl), (de) - 32-bit numbers
Output:
(de) = (de) - (hl)
Destroyed:
none

sbc32

Subtract two 32-bit numbers and the carry bit

Input:
(hl), (de) - 32-bit numbers
Output:
(de) = (de) - (hl) - cf
Destroyed:
none
